# Storefront Template — Features

A pitch-ready list of everything the template ships with, organized by who benefits and ordered to read top-down with prospective clients.

The basics (cart, checkout, products, search, accounts, admin CRUD, SEO, mobile-responsive, RTL Hebrew) are taken as a given — this list focuses on what makes the template **stand out** vs a generic Shopify / WooCommerce setup.

**Live demo:** https://storefront-template-demo.vercel.app

---

## 🛍️ חוויית קונה — מנועי המרה

| | |
|---|---|
| **`Add-to-cart` עם אנימציית טיסה** | תמונת המוצר עפה אל אייקון הסל בהקלקה — דופמין מיידי ב-360ms, יותר זיכרון לאקט הקנייה |
| **קונפטי בהשלמת הזמנה** | פיצוץ בז׳-זהב בצבעי המותג בעמוד "תודה" — חוויה חגיגית במקום דף עיוור |
| **Free-shipping progress bar** | "עוד ₪50 ויש לך משלוח חינם 🎁" בעגלה — מגדיל ערך הזמנה ממוצע ב-20-30% במחקרים |
| **Recently viewed strip** | רצועת מוצרים שראית לאחרונה אוטומטית — שמירה ב-`localStorage`, גלילה אופקית במובייל |
| **Restock waitlist** | "מוצר חזר למלאי? תודיעו לי" — לקוחה משאירה אימייל, מקבלת מייל אוטומטי ברגע שהמוצר חזר |
| **גלילה אופקית במובייל** | מוצרים וקטגוריות בדף הבית = swipe ימינה-שמאלה כמו פיד אינסטגרם, במקום sccroll אנכי משעמם |
| **Wishlist** | שמירת מוצרים מועדפים עם sync בין סשנים ועמוד ייעודי |
| **חיפוש חכם** | בכותרת — חיפוש בכל המוצרים, הקטגוריות והתתי-קטגוריות |

## 🗂️ מבנה תוכן גמיש

| | |
|---|---|
| **תתי-קטגוריות מקוננות** | קטגוריה ראשית (כמו "אביזרים") יכולה להכיל תתי-קטגוריות (כמו "תיקים / כובעים / חגורות") — 2 רמות עומק |
| **`Dropdown` בנאב** | ריחוף על קטגוריה-אב בנאב הראשי = `dropdown` עם תתי-קטגוריות, בלי קליק |
| **גרירה לסידור — תתי-קטגוריות + עליון** | האדמין גורר עכבר/אצבע לסדר קטגוריות וכותב לשרת אוטומטית. גם תתי-קטגוריות מתוך טופס ההורה |
| **`Featured products`** | סימון מוצרים שיופיעו בדף הבית — בקליק אחד מהרשימה |
| **קטגוריות "ראשי" באוטומציה** | בלי תמונה לקטגוריה? פאלבק לגרדיאנט מבוסס מותג — אתר אף פעם לא נראה ריק |

## ⚡ ניהול חנות יעיל (Admin)

| | |
|---|---|
| **`Bulk product editor`** | בחירה מרובה של מוצרים → סרגל פעולות עליון: שינוי מחיר באחוז (`20%-` למבצע!), עדכון מלאי באצווה, הצגה/הסתרה, סימון כמוצג, העברה לקטגוריה, מחיקה. כולל **מודאל תצוגה מקדימה** לפני ביצוע |
| **טיפול חכם בתתי-קטגוריות** | טופס קטגוריה כולל סקשן "תתי-קטגוריות" — הוספה/עריכה/גרירה/מחיקה inline, שמירה אחת לכל ההיררכיה |
| **`Product import wizard`** | יבוא כמות מוצרים מקובץ `CSV` של `Shopify` / `WooCommerce` ב-3 צעדים: העלאה → תצוגה מקדימה → ייבוא בbatches. כולל **הורדת תמונות מקור ל-Vercel Blob** כדי שיהיו על האחסון של הלקוח |
| **`Customer segments` אוטומטיים** | כל לקוחה מתויגת אוטומטית: VIP / חוזרת / חדשה / בסיכון / נטושה — לפי דפוסי קנייה. עוזר לכוון מבצעים נכון |
| **`LTV` לכל לקוחה** | ערך לקוחה לאורך החיים — ההכנסה הכוללת מכל הקנייות שלה, מסומן ברשימת לקוחות |
| **`Order workflow` עם סטטוסים** | משלוח / טיפול / נשלח / נמסר — שינוי סטטוס מעדכן הזמנה + נשמר ב-DB עם היסטוריה |
| **`Coupons`** | הנחות באחוז, סכום קבוע או משלוח חינם. הגבלות שימוש, תוקף, רמת לקוחה. כולל קוד מותאם אישית |
| **`Popups campaigns`** | יצירת פופאפים באדמין: ניוזלטר / מבצע / הודעה. **הגדרת טריגרים** (אחרי X שניות / גלילה ל-Y / יציאה מהדף), תדירות, תאריך התחלה+סיום, **תצוגה מקדימה חיה** באדמין |
| **`Abandoned cart recovery`** | מערכת אוטומטית לזיהוי עגלות נטושות + שליחת מייל תזכורת (דורש הגדרת `Resend`) |
| **`Reports dashboard`** | הכנסות, מוצרים מובילים, קטגוריות מובילות, גרפים לאורך זמן |
| **`Command palette` (`Cmd+K`)** | קיצור מקלדת באדמין — קפיצה מהירה לכל עמוד / מוצר / לקוחה |
| **תצוגה מקובצת של תתי-קטגוריות** | רשימת הקטגוריות באדמין — תתי-הקטגוריות מסודרות יפה בפאנל ייעודי מתחת לאב, לא מתפזרות בטבלה |

## 🎨 מערכת עיצוב — `White-label` מלא

| | |
|---|---|
| **קובץ אחד מחליף הכל** | פלטת צבעים + לוגו + שם מותג + פרטי קשר → 3 קבצים בלבד (`siteConfig.ts`, `globals.css`, `/public/brand/`) |

| **תמונות placeholder אוטומטיות** | אתר חדש נראה מוכן: לוגו טקסט, hero בגרדיאנט, og-image — מ-יום 1, לא ריק |

## 🛠️ תשתית טכנית

| | |
|---|---|
| **`Next.js 16` + `App Router` + `Turbopack`** | טעינה מהירה, `RSC` כברירת מחדל, `streaming` |

| **`Vercel Blob`** | אחסון תמונות מוצרים — מוכן לחיבור |
| **`Resend`** | מערכת מיילים טרנזקציונליים — מוכן לחיבור |
| **`Mock payment`** | מערכת תשלום מדומה לפיתוח + הדגמה — מחליפים ל-PSP אמיתי בקלות |

## ♿ נגישות + ביצועים

- **`prefers-reduced-motion`** — אנימציות (קונפטי, fly-to-cart) מכבדות העדפת המשתמש
- **`aria-label` בכל אינטראקציה** — תומך קוראי-מסך
- **תמיכת מקלדת מלאה** — כולל ב-drag-and-drop
- **לוגיקה ב-RSC** — מינימום JS ללקוח
- **`<Image>` של Next** — אופטימיזציה אוטומטית
- **`tabular-nums` בכל המספרים** — קריאה נוחה של מחירים וכמויות

## 🎯 מה אין במתחרים

קומבינציה של הפיצ׳רים הבאים — לא תמצאי כולם בחבילה אחת בשום פלטפורמה מתחרה בישראל:

1. **`White-label` מלא של האדמין** (לא רק החנות)
2. **תתי-קטגוריות עם hover-dropdown** + עריכה inline + גרירה
3. **`Bulk product editor` עם תצוגה מקדימה** (לא רק "select + delete")
4. **גלילה אופקית native במובייל** (לא scroll-snap מזויף)
5. **Restock waitlist אוטומטית** עם מייל מותאם מותג
6. **קונפטי + fly-to-cart** — חוויית קנייה שמרגישה כמו אפליקציה, לא אתר
7. **קוד מקור פתוח** — אפשר להוסיף בלי הגבלת פלטפורמה (לא Shopify lock-in)
8. **תמיכת RTL מקצועית** — לא תרגום מ-LTR
9. **Vercel + Neon — scale to zero** — לא משלמים כשאין מבקרים
